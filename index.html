# 🧠 Smart Flashcards Pro (v3.0) 完全開発・設計ログ - [[App_Development_Log]] - **作成日:** 2026-06-28 - **ステータス:** バージョン3.0 最終安定版（正答率モデル・Attempt_Count対応・苦手優先ソート刷新） - **動作検証済み環境:** iOS 17+ (Safari / PWA / 各種アプリ内ブラウザ), Windows 10/11 (Chrome / Edge), macOS (Safari / Chrome) - **システム構成:** Frontend (HTML5/Tailwind/JS) + Backend (GAS Web App) + Database (Google Sheets) ## 1. アプリケーション概要 本プロジェクトは、Googleスプレッドシートを完全にセキュアで編集しやすいパーソナルデータベースとして活用する暗記カードシステム（PWA）の発展型である。 バージョン3.0では、学習データ分析の基礎である「単に正答数が多いか少ないか」を脱却し、**「その問題を何回解いて（Attempt）、何回正解したか（Correct）」から導き出す「正答率（Accuracy Rate）」ベースでの進捗追跡＆苦手抽出モデル**を構築した。 子供が自作問題のテキスト入力や画像の登録を容易に行えるスプレッドシートの手軽さを損なわないため、**正答率の計算やソートの判断ロジックはすべてアプリ内（フロントエンド）に寄せる設計**とし、数式エラーの発生しない堅牢な構造に進化させている。 ## 2. v2.0（前作）からの主要な進化・変更点 ### ① 「実施回数（`Attempt_Count`）」カラムの追加 - **データスキーマ拡張:** 各学習シート（タブ）の `Correct_Count` カラムの前に、新たに `Attempt_Count` 列を挿入可能に。 - **未学習問題（未入力・0回）の最適化（案①の採用）:** 新規作成問題など、まだ1度も解いておらず `Attempt_Count` が空欄、または0回となっているカードは、プログラム内で**例外的に「正答率 0%」**として扱う。これにより、苦手優先モードに切り替えた時に「新しく追加した問題が必ず最優先で並ぶ」学習体験を実現。 ### ② 「正答率」統計表示への刷新 - **直観的な進捗確認:** これまでの単なる「正解数：X回」の表示を廃止し、**「正答率：XX% (正答数/実施回数)」** の形に画面下部のUI表示を刷新。一目でその問題がどの程度身についているかを定量的に把握可能にした。 ### ③ 正答率ベースの「苦手優先」ソートエンジンの刷新 - **多段階（フォールバック）ソート設計:** 1. **第一基準（正答率）：** 正答率（$Correct\_Count / Attempt\_Count$）が「低い」問題から順に並べる（未学習=0% が最優先）。 2. **第二基準（挑戦回数）：** 正答率が同じ場合（例：どちらも0%）、より挑戦していない問題（`Attempt_Count` が少ない＝0回のもの）を前に配置する。 3. **第三基準（新しさ）：** 完全に同格の場合は、`ID` の降順（新しい順）にする。 この多段階処理により、単にランダムに並ぶのとは違い、「新問や本当に苦手な問題」を効率的に反復できるようになる。 ### ④ 評価書き込み時のGAS連動（v3.0 向けGAS拡張） - ユーザーが「覚えた！」または「違った」をタップした際、アプリからGAS側へ同期通信を送信。 - GAS側は、**「間違えたときには、Attempt_Count（実施回数）のみを+1する」「正解したときには、Attempt_Count と Correct_Count の両方を+1する」**という新たなデータ更新ルールを実装。 --- ## |**B列**|`Category`|文字列|予備分類。| |**C列**|`Problem`|文字列|問題文。| |**D列**|`Problem_voice`|文字列|問題の読み（日本語誤読防止用）。| |**E列**|`Answer`|文字列|解答（カード裏面最上部）。| |**F列**|`Answer_voice`|文字列|解答の読み。| |**G列**|`Explanation`|文字列|補足解説。| |**H列**|`URL`|文字列|補足用画像URL。| |**I列**|`Attempt_Count`|数値|**実施回数（新設・v3.0）**。パス以外の解答アクション時に必ず `+1` されます（空欄スタートでOK）。| |**J列**|`Correct_Count`|数値|**正答回数の記録**。「覚えた！」を押すと自動で
